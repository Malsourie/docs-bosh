# delete_stemcell

Deletes previously created stemcell. Assume that none of the VMs require presence of the stemcell.


## Arguments

 * `stemcell_cid` [String]: Cloud ID of the stemcell to delete; returned from `create_stemcell`.


## Result

No return value
